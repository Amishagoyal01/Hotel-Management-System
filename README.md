# 🏨 Hotel Management System (HMS)  

Welcome to the **Hotel Management System** project! 🎉  
This Python 🐍 console application helps in **Room, Hall, Table bookings** along with **Food Orders 🍲🥤🍰**.  
All customer details and orders are stored in **dictionaries 📂**.  

---

## 🌟 Features  

| 🔢 Option | 📝 Operation | 💡 Description |
|-----------|--------------|----------------|
| 1️⃣ | 🛏️ **Bookings** | Book rooms, halls, or tables with customer details |
| 2️⃣ | 🍽️ **Food Orders** | Order food (Veg, Non-Veg, Drinks, Desserts) as existing or new customer |
| ✅ | 💰 **Bill Management** | All booking and food bills are added & shown clearly |
| 🔄 | 👥 **Existing/New Users** | Supports both new customer registrations & existing customers with Booking ID / Order ID |

---

## 🛠️ Technology Used  

| 🔧 Tool | 💡 Purpose |
|---------|------------|
| 🐍 Python | Core programming language |
| 📂 Dictionaries | Store customer details, bookings & food orders |
| 🎲 Random Module | Auto-generate Booking IDs & Order IDs |

---

## 📋 How It Works  

| Step | Action |
|------|---------|
| 1️⃣ | Run the program ▶️ |
| 2️⃣ | Choose **Bookings** 🛏️ or **Food Orders** 🍽️ |
| 3️⃣ | If **Bookings** → Select `Existing User` 🔑 or `New User` 🆕 |
| 4️⃣ | New user provides **Name + Phone Number** 📱 → System generates **Booking ID** 🎲 |
| 5️⃣ | Choose type of booking:<br>• 🛏️ Room<br>• 🏢 Hall<br>• 🍽️ Table |
| 6️⃣ | After booking, system asks if you want to **order food** 🍲 |
| 7️⃣ | Food Orders can be placed by:<br>• Existing Food Customer (Order ID 🔑)<br>• New Food Customer (Name + Phone Number 📱 → System generates **Order ID**) |
| 8️⃣ | Choose from **Veg, Non-Veg, Drinks, Desserts** menus |
| 9️⃣ | ✅ Bills keep updating in dictionaries |
| 🔟 | At the end, **Total Bill** 💰 is displayed |
